Python

Python é uma linguagem de programação interpretada, orientada a objetos e de alto nível com semântica dinâmica. Sendo utilizada para desenvolvimento web do lado do servidor, desenvolvimento de software e scripts de sistemas.

Django

Django é uma framework web para Python, que incentiva o desenvolvimento rápido e um design limpo e pragmático. Ele é responsável por grande parte do trabalho de desenvolvimento web.

JavaScript

JavaScript é uma linguagem leve e habitualmente usado nas páginas webs, cujas implementações permitem que o script do lado do cliente interaja com o usuário e crie páginas dinâmicas. É uma linguagem de programação interpretada com recursos orientados a objetos.

TypeScript

TypeScript é uma linguagem de programação desenvolvida e mantida pela Microsoft. Apresentando recursos adicionais, como vinculação de tipos estritos para JavaScript. TypeScript supera uma das maiores falhas do JavaScript, que é encontrar bugs no código durante a fase de desenvolvimento, verificando qualquer problema no tempo de compilação, impedindo a entrega do produto com defeitos para o cliente final.

GitHub

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git, permitindo que os programadores ou qualquer usuário da plataforma contribuam em projetos privados e/ou Open Source em qualquer lugar do mundo.

Bootstrap

Bootstrap é um framework de desenvolvimento front-end criado e mantido pelo Twitter. Oferecendo uma ampla gama de componentes e ferramentas, como uma estrutura de grid responsiva e um conjunto de estilos pré-construídos, o Bootstrap facilita a criação de sites modernos e compatíveis com diferentes dispositivos. Ele resolve um dos maiores desafios no desenvolvimento web, que é garantir que o design funcione de maneira consistente em diversas plataformas e tamanhos de tela, permitindo a entrega de uma experiência de usuário fluida e bem estruturada para o cliente final.

PHP

PHP é uma linguagem de programação amplamente usada no desenvolvimento web, conhecida por sua capacidade de criar conteúdo dinâmico e interagir com bancos de dados. Desenvolvida por Rasmus Lerdorf, ela facilita a integração com várias tecnologias e permite uma rápida implementação e ajuste de código, o que ajuda a entregar aplicativos web com menos defeitos e maior qualidade.

Postgresql

PostgreSQL é um sistema de gerenciamento de banco de dados relacional de código aberto, conhecido por sua robustez e extensibilidade. Ele oferece suporte a recursos avançados como transações ACID e consultas complexas, e permite a personalização através de extensões e funções definidas pelo usuário. É ideal para gerenciar grandes volumes de dados e garantir a integridade e segurança das informações.

Oracle

Oracle é um sistema de gerenciamento de banco de dados relacional desenvolvido pela Oracle Corporation, conhecido por sua escalabilidade, segurança e desempenho. Amplamente utilizado em grandes empresas, oferece funcionalidades avançadas como suporte a transações ACID, replicação de dados e ferramentas de análise. É uma solução confiável para gerenciar grandes volumes de dados e aplicações críticas.

Docker

Docker consiste em uma plataforma Open Source, cujo objetivo é gerenciar aplicativos em contêineres como se fossem VM’s. Os contêineres são isolados uns dos outros e agrupam seus próprios softwares, bibliotecas e arquivos de configuração.
